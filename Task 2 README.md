# AI Personal Assistant Chatbot

## Overview

The **AI Personal Assistant Chatbot** is a Python-based interactive chatbot application designed to provide automated responses and assist users through a simple conversational interface. The project implements chatbot functionalities such as user interaction, prompt templates, intelligent responses, and chat history management.

The chatbot allows users to select predefined prompt templates for tasks such as summarizing text, writing emails, explaining topics, and generating ideas. User conversations are stored and saved into a text file for future reference. The project demonstrates the practical implementation of Python programming concepts including functions, loops, conditional statements, dictionaries, lists, and file handling.

## Working Principle

1. The chatbot starts by displaying a welcome message.
2. Users select a prompt template or directly enter a question.
3. The user input is processed by the response generation function.
4. The chatbot provides a suitable response.
5. Conversations are stored in the chat history.
6. The chat history is saved as a text file after completion.
