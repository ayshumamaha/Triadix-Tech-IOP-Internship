# AI Workflow Automation Platform

## Overview

AI Workflow Automation Platform is a simple AI-based application that demonstrates how multiple AI agents can work together to automate a workflow. The platform accepts a user task and processes it through different agents to produce organized results.

---

## Objectives

- Understand workflow automation
- Learn multi-agent architecture
- Automate sequential tasks
- Explore FastAPI backend development
- Understand local LLM integration concepts
- Build a simple AI workflow platform

---

## Features

- Multi-Agent Workflow
- Task Automation
- FastAPI Backend
- Local LLM Ready (Placeholder)

- Simple Workflow Execution

## Workflow

```
User Task
     ↓
Planner Agent
     ↓
Research Agent
     ↓
Summarizer Agent
     ↓
Final Output
```

---

## Agents

### Planner Agent
- Receives the user task.
- Plans the workflow.

### Research Agent
- Performs the research step.
- Processes the planned task.

### Summarizer Agent
- Generates the final summarized output.

---

## How It Works

1. User enters a task.
2. Planner Agent plans the task.
3. Research Agent processes the task.
4. Summarizer Agent creates the final summary.
5. Results are displayed to the user.
