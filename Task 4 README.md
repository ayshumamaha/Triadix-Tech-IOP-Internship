# AI PDF Knowledge Assistant

## Overview

AI PDF Knowledge Assistant is a simple Retrieval-Augmented Generation (RAG)-based application that allows users to upload multiple PDF documents, search them using semantic similarity, and retrieve relevant information with source citations.

The project uses sentence embeddings and a FAISS vector database to perform intelligent document retrieval.

---

## Objectives

- Understand Retrieval-Augmented Generation (RAG)
- Learn document embeddings
- Store vectors using FAISS
- Perform semantic search
- Retrieve relevant information from PDFs
- Build a document-based AI application

---

## Features

- Upload Multiple PDF Documents
- Extract Text from PDFs
- Semantic Search
- Question Answering
- Citation Support
- Multiple Document Support
- FAISS Vector Database
- Sentence Transformer Embeddings

---

## Technologies Used

- Python
- Google Colab
- PyPDF
- Sentence Transformers
- FAISS
- NumPy

---

## Project Workflow

1. Upload one or more PDF documents.
2. Extract text from each PDF.
3. Generate embeddings using Sentence Transformers.
4. Store embeddings in a FAISS vector database.
5. Enter a question.
6. Perform semantic search.
7. Retrieve the most relevant content.
8. Display the source PDF and page number.

---

## Sample Questions

- What is Artificial Intelligence?
- Explain supervised learning.
- What are Large Language Models?
- Explain Retrieval Augmented Generation.
- What are the applications of AI?

## Output

The system displays:
