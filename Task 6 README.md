# AI Learning Copilot

## Overview

AI Learning Copilot is an AI-powered learning assistant that helps users study more effectively by allowing them to upload PDF documents, ask questions, generate quizzes and flashcards, create study plans, and monitor learning progress. The project uses semantic search and vector embeddings to retrieve relevant information from uploaded study materials.

---

## Objectives

- Build an AI-powered learning assistant
- Understand document processing and semantic search
- Learn vector embeddings using Sentence Transformers
- Implement a searchable knowledge base using FAISS
- Generate learning resources from study materials
- Create a personalized study planner

---

## How It Works

1. Upload one or more PDF study materials.
2. Extract text from each document.
3. Convert the text into vector embeddings.
4. Store embeddings in a FAISS vector database.
5. Ask questions related to the uploaded documents.
6. Retrieve the most relevant content using semantic search.
7. Generate quizzes and flashcards.
8. Display a weekly study plan.
9. Show learning progress.
